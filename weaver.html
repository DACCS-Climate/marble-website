Weaver

Weaver "moves the application to the data."

By extending OGC API Processes (Open Geospatial Consortium) to execute computing processes and retrieve metadata surrounding images, Weaver cuts down data transfer sizes and wait times.

Weaver allows the creation of workflows that span several applications by linking these applications to the inputs and outputs of Web Processing Services.

Remote Sensing Toolbox

Nov 11

This is a wider card with supporting text below as a natural lead-in to additional content.

Weaver extends the Open Geospatial API processes to enable the execution of computing processes and the retrieval of metadata describing their purpose and functionality. Typically this data consists of raster, vector, and coverage information.

Several satellite images can be combined into a single data product. This stack of images can then be used in analyses such as Advanced Remote Sensing where the analysis can be performed on a variety of images.
