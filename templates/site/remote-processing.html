Finch is the go-to for climate indicator analysis. With many pre-programmed processes for single indicator analysis and ensemble analysis of a particular indicator, Finch covers the breadth and depth needed for climate indicator analysis. Analyze freeze-thaw cycles using the provided Daily Freeze-Thaw indicator process or determine the longest number of consecutive days where daily precipitation below a given threshold using the provided Ensemble GridPoint Process.

Finch also provides data processing support by providing processes to help manage your data, such as converting a geospatial series to a CF-compliant netCDF.

Raven is an open source server project offering physiographic analysis capabilities through the Web Processing Service (WPS) standard. The outputs of these processes can be used to help configure the Raven hydrological modeling framework over a given watershed.

Processes offered include returning the watershed upstream from an outlet, subsetting raster fields over a given watershed, computing watershed properties (area, centroid location, perimeter, gravelius shape index), analyzing terrain characterisics from a digital elevation model (DEM), extracting land-use properties, and computing statistics over a region.

Weaver is an Application, Deployment, and Execution Service (ADES) and an Execution Management Service (EMS). One of its standout features is the ability to facilitate remote execution of processes from many different applications in a workflow chain. This allows climate researchers to harness the capabilities of the Marble network while freeing up their own resources.

With Weaver, there's no need for users to grapple with the intricacies of software installation and management. Instead, researchers can seamlessly access software residing on the Marble network and transmit precise analysis instructions via Weaver, enabling them to perform data analysis on the extensive datasets stored on Marble network servers.

In essence, Weaver acts as a bridge between researchers and the computational resources they need, simplifying workflow execution and enhancing the efficiency of climate research endeavors.
